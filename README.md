# Sales Demand Forecasting

## Overview
An AI-powered Sales Demand Forecasting web application built using Flask, Python, Scikit-Learn, Plotly, Bootstrap, and Machine Learning.

## Features
- Upload CSV dataset
- Automatic data preprocessing
- Train ML model
- Sales prediction
- Interactive dashboards
- Feature Importance
- Actual vs Predicted graph
- Download predictions

## Technologies
- Python
- Flask
- Pandas
- Scikit-Learn
- Plotly
- Bootstrap
- HTML/CSS/JavaScript

## Installation

```bash
pip install -r requirements.txt
python app.py
